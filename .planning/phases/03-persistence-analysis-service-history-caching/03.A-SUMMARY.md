## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Python 3.9 type annotation compatibility**
- **Found during:** Task 5 (verification step)
- **Issue:** `Mapped[str | None]` union syntax fails on Python 3.9 because SQLAlchemy de-stringifies annotations at runtime, and `str | None` is a Python 3.10+ syntax
- **Fix:** Replaced all `X | None` with `Optional[X]` in models.py and engine.py, using `typing.Optional`, `typing.Dict`, `typing.List`
- **Files modified:** backend/src/db/models.py, backend/src/db/engine.py
- **Verification:** `python3 -c "from src.db.models import Base, User, ApiKey, Analysis, Job, UsageEvent; print('ORM OK')"` passes
