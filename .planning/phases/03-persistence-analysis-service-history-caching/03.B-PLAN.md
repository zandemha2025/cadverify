**Hashing utilities:**
```python
import hashlib

def compute_mesh_hash(file_bytes: bytes) -> str:
    """SHA-256 hex digest of raw uploaded file bytes (D-09)."""
    return hashlib.sha256(file_bytes).hexdigest()

def compute_process_set_hash(process_values: list[str]) -> str:
    """SHA-256 of sorted, comma-joined process type values (D-11)."""
    canonical = ",".join(sorted(process_values))
    return hashlib.sha256(canonical.encode()).hexdigest()
```

**Cache lookup:**
```python
async def _check_cache(
    session: AsyncSession,
    user_id: int,
    mesh_hash: str,
    process_set_hash: str,
    analysis_version: str,
) -> Analysis | None:
    stmt = select(Analysis).where(
        Analysis.user_id == user_id,
        Analysis.mesh_hash == mesh_hash,
        Analysis.process_set_hash == process_set_hash,
        Analysis.analysis_version == analysis_version,
    )
    return (await session.execute(stmt)).scalars().first()
```

**Persist analysis:**
```python
async def _persist_analysis(
    session: AsyncSession,
    user: AuthedUser,
    mesh_hash: str,
    process_set_hash: str,
    analysis_version: str,
    filename: str,
    file_type: str,
    file_size_bytes: int,
    result_json: dict,
    verdict: str,
    face_count: int,
    duration_ms: float,
) -> Analysis:
    analysis = Analysis(
        ulid=str(ULID()),
        user_id=user.user_id,
        api_key_id=user.api_key_id,
        mesh_hash=mesh_hash,
        process_set_hash=process_set_hash,
        analysis_version=analysis_version,
        filename=filename,
        file_type=file_type,
        file_size_bytes=file_size_bytes,
        result_json=result_json,
        verdict=verdict,
        face_count=face_count,
        duration_ms=duration_ms,
    )
    session.add(analysis)
    await session.flush()  # get id assigned
    return analysis
```

**Main entry points:**
```python
async def run_analysis(
    file_bytes: bytes,
    filename: str,
    processes: str | None,
    rule_pack: str | None,
    user: AuthedUser,
    session: AsyncSession,
) -> dict:
```
This function:
1. Computes `mesh_hash = compute_mesh_hash(file_bytes)`
2. Resolves target processes via `_resolve_target_processes(processes)` (import from routes or extract)
3. Computes `process_set_hash = compute_process_set_hash([p.value for p in proc_list])`
4. Reads `analysis_version` from `src.__version__`
5. Calls `_check_cache(session, user.user_id, mesh_hash, process_set_hash, analysis_version)`
6. On cache HIT: calls `_write_usage_event(session, user, "analysis_cached", cached.id, mesh_hash, cached.duration_ms, cached.face_count)` and returns `cached.result_json`
7. On cache MISS:
   a. Calls `_parse_mesh(file_bytes, filename)` (import from routes)
   b. Runs the full pipeline in threadpool (same pattern as current `validate_file`)
   c. Calls `_to_response(result, features, pack)` to get the result dict
   d. Calls `_persist_analysis(...)` with the result dict
   e. Calls `_write_usage_event(session, user, "analysis_complete", analysis.id, mesh_hash, duration_ms, face_count)`
   f. Returns result dict
8. On IntegrityError (concurrent duplicate): catches `sqlalchemy.exc.IntegrityError`, re-queries cache, returns existing row
