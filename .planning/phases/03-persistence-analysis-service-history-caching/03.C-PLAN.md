Create `backend/src/api/history.py` with a new `APIRouter(prefix="/analyses", tags=["history"])`:

**GET /analyses (list):**
```python
@router.get("")
@limiter.limit("60/hour;500/day")
async def list_analyses(
    request: Request,
    response: Response,
    cursor: str | None = Query(None, description="ULID cursor for pagination"),
    limit: int = Query(20, ge=1, le=100, description="Results per page (max 100)"),
    verdict: str | None = Query(None, description="Filter: pass, issues, fail"),
    user: AuthedUser = Depends(require_api_key),
    session: AsyncSession = Depends(get_db_session),
):
```

Query logic:
1. `stmt = select(Analysis).where(Analysis.user_id == user.user_id)`
2. If `cursor`: `stmt = stmt.where(Analysis.ulid < cursor)`
3. If `verdict` and verdict in ("pass", "issues", "fail"): `stmt = stmt.where(Analysis.verdict == verdict)`
4. `stmt = stmt.order_by(Analysis.created_at.desc()).limit(limit + 1)`
5. Execute, check if len(results) > limit for has_more
6. Build summary response per D-20:
   ```python
   {
       "analyses": [
           {
               "id": a.ulid,
               "filename": a.filename,
               "file_type": a.file_type,
               "verdict": a.verdict,
               "face_count": a.face_count,
               "duration_ms": a.duration_ms,
               "created_at": a.created_at.isoformat(),
               "process_count": len(a.result_json.get("process_scores", [])),
               "best_process": a.result_json.get("best_process"),
           }
           for a in items
       ],
       "next_cursor": items[-1].ulid if has_more else None,
       "has_more": has_more,
   }
   ```

**GET /analyses/{analysis_id} (detail):**
```python
@router.get("/{analysis_id}")
@limiter.limit("60/hour;500/day")
async def get_analysis(
    analysis_id: str,
    request: Request,
    response: Response,
    user: AuthedUser = Depends(require_api_key),
    session: AsyncSession = Depends(get_db_session),
):
```

Query logic:
1. `stmt = select(Analysis).where(Analysis.ulid == analysis_id, Analysis.user_id == user.user_id)`
2. If not found: raise HTTPException(404, detail="Analysis not found")
3. Return per D-21:
   ```python
   {
       "id": analysis.ulid,
       "filename": analysis.filename,
       "file_type": analysis.file_type,
       "created_at": analysis.created_at.isoformat(),
       "result": analysis.result_json,
   }
   ```

Import logger as: `logger = logging.getLogger("cadverify.history")`
