<action>
Create `backend/src/services/share_service.py` with:

1. `BASE62_CHARS = string.digits + string.ascii_letters` (62 characters: 0-9A-Za-z)

2. `generate_short_id(length: int = 12) -> str`:
   - Uses `secrets.token_bytes(length)` for cryptographic randomness
   - Maps each byte to `BASE62_CHARS[byte % 62]`
   - Returns 12-char string

3. `async def create_share(analysis_ulid: str, user_id: int, session: AsyncSession) -> dict`:
   - SELECT analysis WHERE ulid=analysis_ulid AND user_id=user_id
   - If not found: raise HTTPException(404, "Analysis not found")
   - If already shared (share_short_id is not None): return existing share URL
   - Generate short_id via generate_short_id()
   - UPDATE analysis SET share_short_id=short_id, is_public=True
   - Return `{"share_url": f"/s/{short_id}", "share_short_id": short_id}`

4. `async def revoke_share(analysis_ulid: str, user_id: int, session: AsyncSession) -> None`:
   - SELECT analysis WHERE ulid=analysis_ulid AND user_id=user_id
   - If not found: raise HTTPException(404, "Analysis not found")
   - UPDATE analysis SET share_short_id=None, is_public=False

5. `async def get_shared_analysis(short_id: str, session: AsyncSession) -> dict | None`:
   - SELECT analysis WHERE share_short_id=short_id AND is_public=True
   - If not found: return None
   - Return sanitize_analysis_for_share(analysis)

6. `def sanitize_analysis_for_share(analysis: Analysis) -> dict`:
   - Explicit allow-list of fields: filename, file_type, verdict, face_count, duration_ms, created_at (ISO format)
   - From result_json: process_scores, universal_issues, geometry, best_process, priority_fixes
   - EXCLUDES: user_id, api_key_id, email, mesh_hash, share_short_id, is_public, ulid
   - Returns flat dict
</action>

   ```typescript
     const res = await fetch(`${API_BASE}/analyses/${id}/share`, {
       method: "POST",
       headers: authHeaders(),
     });
     if (!res.ok) {
       const err = await res.json().catch(() => ({ detail: res.statusText }));
       throw new Error(err.detail || "Failed to share analysis");
     }
     return res.json();
   }
   ```

   ```typescript
   export async function unshareAnalysis(id: string): Promise<void> {
     const res = await fetch(`${API_BASE}/analyses/${id}/share`, {
       method: "DELETE",
       headers: authHeaders(),
     });
     if (!res.ok) {
       const err = await res.json().catch(() => ({ detail: res.statusText }));
       throw new Error(err.detail || "Failed to unshare analysis");
     }
   }
   ```
