Create `backend/src/services/pdf_service.py` with:

```python
import asyncio
import logging
import os
from pathlib import Path

from jinja2 import Environment, FileSystemLoader
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from src.db.models import Analysis

logger = logging.getLogger("cadverify.pdf")

PDF_CACHE_DIR = os.getenv("PDF_CACHE_DIR", "/data/pdf-cache/")
_TEMPLATE_DIR = Path(__file__).resolve().parent.parent / "templates" / "pdf"

_pdf_semaphore = asyncio.Semaphore(2)

_jinja_env = Environment(loader=FileSystemLoader(str(_TEMPLATE_DIR)))
```

1. **Custom Jinja2 filters:**
   - `format_duration(ms)` -> "1.23s" or "0.5s"
   - `severity_order(severity)` -> 0 for error, 1 for warning, 2 for info (for sorting)

2. **`def _render_pdf_sync(context: dict) -> bytes`:**
   - Loads `analysis_report.html` template from _jinja_env
   - Renders HTML string with context
   - Calls `HTML(string=html_str, base_url=str(_TEMPLATE_DIR)).write_pdf()`
   - Returns PDF bytes
   - This is synchronous (WeasyPrint is CPU-bound)

3. **`async def generate_pdf(analysis: Analysis) -> bytes`:**
   - Builds context dict: filename, file_type, verdict, face_count, duration_ms, created_at (ISO), result (result_json), engine_version (from `src.__version__`), mesh_hash (first 12 chars of analysis.mesh_hash)
   - Acquires `_pdf_semaphore`
   - Runs `_render_pdf_sync(context)` via `asyncio.get_event_loop().run_in_executor(None, ...)`
   - Returns PDF bytes

4. **`async def get_or_generate_pdf(analysis_ulid: str, user_id: int, session: AsyncSession) -> tuple[bytes, str]`:**
   - SELECT analysis WHERE ulid=analysis_ulid AND user_id=user_id
   - If not found: raise HTTPException(404, "Analysis not found")
   - Validate: assert "/" not in analysis.ulid and ".." not in analysis.ulid
   - cache_path = Path(PDF_CACHE_DIR) / f"{analysis.ulid}.pdf"
   - If cache_path.exists(): return (cache_path.read_bytes(), analysis.filename)
   - Else: generate PDF, mkdir parents, write to cache_path, return (pdf_bytes, analysis.filename)
   - Returns tuple of (pdf_bytes, original_filename) for Content-Disposition header

5. **`def _safe_filename(filename: str) -> str`:**
   - Strips unsafe chars from filename for Content-Disposition
   - Returns `{stem}-dfm-report.pdf` where stem is the original filename without extension
</action>

```typescript
export async function downloadPdf(analysisId: string, filename: string): Promise<void> {
  const res = await fetch(`${API_BASE}/analyses/${analysisId}/pdf`, {
    headers: authHeaders(),
  });
  if (!res.ok) {
    const err = await res.json().catch(() => ({ detail: res.statusText }));
    throw new Error(err.detail || "Failed to download PDF");
  }
  const blob = await res.blob();
  const stem = filename.replace(/\.[^.]+$/, "");
  const downloadName = `${stem}-dfm-report.pdf`;
  
  // Create temporary download link
  const url = URL.createObjectURL(blob);
  const a = document.createElement("a");
  a.href = url;
  a.download = downloadName;
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
  URL.revokeObjectURL(url);
}
```

This fetches the PDF as a blob, creates a temporary anchor element to trigger the browser download with the correct filename, then cleans up.
</action>
