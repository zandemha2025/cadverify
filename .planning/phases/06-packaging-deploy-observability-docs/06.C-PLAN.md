<task id="06C-1">
<title>Expand .github/workflows/ci.yml with migration check, Docker build, typecheck</title>
<read_first>
- .github/workflows/ci.yml
- backend/Dockerfile
- backend/fly.toml
</read_first>
<action>
Update `.github/workflows/ci.yml` to add the following:

**1. Add Alembic migration check to backend job:**
After the "Run tests" step, add:

```yaml
      - name: Alembic migration check
        run: |
          pip install -r requirements.txt
          python -c "
          from sqlalchemy import create_engine
          engine = create_engine('sqlite:///test_migrations.db')
          "
          DATABASE_URL=sqlite:///test_migrations.db alembic upgrade head
          DATABASE_URL=sqlite:///test_migrations.db alembic downgrade -1
          DATABASE_URL=sqlite:///test_migrations.db alembic upgrade head
          rm -f test_migrations.db
```

**2. Add typecheck step to backend job:**
After the migration check, add:

```yaml
      - name: Typecheck
        run: |
          pip install pyright
          pyright src/ --pythonversion 3.12
        continue-on-error: true
```

(continue-on-error: true initially since existing code may have type issues; remove once clean)

**3. Add docker-build job:**

```yaml
  docker-build:
    name: Docker Build
    runs-on: ubuntu-latest
    needs: [backend]
    permissions:
      contents: read
    steps:
      - uses: actions/checkout@v4

      - name: Set up Docker Buildx
        uses: docker/setup-buildx-action@v3

      - name: Login to Fly.io Registry
        if: github.ref == 'refs/heads/main' && github.event_name == 'push'
        run: |
          echo "${{ secrets.FLY_API_TOKEN }}" | docker login registry.fly.io -u x --password-stdin

      - name: Build and push
        uses: docker/build-push-action@v5
        with:
          context: backend
          push: ${{ github.ref == 'refs/heads/main' && github.event_name == 'push' }}
          tags: registry.fly.io/cadverify-api:${{ github.sha }},registry.fly.io/cadverify-api:latest
          platforms: linux/amd64
          cache-from: type=gha
          cache-to: type=gha,mode=max
```

**4. Update deploy job to use pre-built image:**

Change the deploy job:
```yaml
  deploy:
    name: Deploy to Fly.io
    runs-on: ubuntu-latest
    needs: [backend, frontend, docker-build]
    if: github.ref == 'refs/heads/main' && github.event_name == 'push'
    steps:
      - uses: actions/checkout@v4

      - uses: superfly/flyctl-actions/setup-flyctl@master

      - name: Deploy backend (pre-built image)
        run: |
          flyctl deploy \
            --config backend/fly.toml \
            --image registry.fly.io/cadverify-api:${{ github.sha }} \
            --env RELEASE=${{ github.sha }}
        env:
          FLY_API_TOKEN: ${{ secrets.FLY_API_TOKEN }}
```

Remove the old `--dockerfile` based deploy. Remove the frontend flyctl deploy (frontend deploys via Vercel GitHub integration, not Fly).
</action>
