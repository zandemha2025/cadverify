# Plan 07.C: Pre-baked Weights + Embedding Cache + Visibility + Fallback

<objective>
Complete the deployment infrastructure for async SAM-3D: pre-bake SAM-2 model weights into the Docker image, configure the embedding cache on persistent storage, ensure worker visibility timeout and ack-on-completion semantics, update Fly config for always-on worker, and add integration tests for the fallback chain.
</objective>

<must_haves>
- SAM-2 Hiera Small weights pre-baked into Docker image at /app/models/
- Embedding cache directory on Fly volume (/data/blobs/sam3d_cache)
- Worker visibility timeout >= 10 min (job_timeout=600 in WorkerSettings)
- Fly.toml worker process has auto_stop_machines disabled
- Graceful fallback to heuristic segmentation tested
- DB engine init/dispose hooks for worker process
- .env.example updated with all SAM-3D and job-related env vars
</must_haves>

### Task 3: Add worker DB engine lifecycle hooks

<read_first>
- backend/src/db/engine.py (current engine setup, get_db_session)
- backend/src/jobs/worker.py (startup/shutdown hooks)
</read_first>

<action>
Update `backend/src/db/engine.py` to export `init_engine()` and `dispose_engine()` functions for the worker process (which doesn't use FastAPI's lifespan):

1. Add `async def init_engine()`:
   ```python
   async def init_engine() -> None:
       """Initialize the async engine. Called by FastAPI lifespan or worker startup."""
       global _engine, _session_factory
       if _engine is not None:
           return  # Already initialized
       url = os.getenv("DATABASE_URL", "")
       _engine = create_async_engine(url, pool_size=5, max_overflow=10)
       _session_factory = async_sessionmaker(_engine, expire_on_commit=False)
   ```

2. Add `async def dispose_engine()`:
   ```python
   async def dispose_engine() -> None:
       """Dispose the async engine. Called by worker shutdown."""
       global _engine, _session_factory
       if _engine is not None:
           await _engine.dispose()
           _engine = None
           _session_factory = None
   ```

3. Export `async_session_factory` as a module-level getter:
   ```python
   def get_session_factory():
       """Get the session factory. Raises if engine not initialized."""
       if _session_factory is None:
           raise RuntimeError("DB engine not initialized. Call init_engine() first.")
       return _session_factory
   ```

4. Update `backend/src/jobs/tasks.py` to use `get_session_factory()` instead of directly importing `async_session_factory`.

Note: Ensure the existing FastAPI `get_db_session` dependency still works -- it should call `init_engine()` lazily if not yet initialized, or rely on FastAPI lifespan to call it.
</action>

- `backend/src/db/engine.py` contains `async def init_engine(`
- `backend/src/db/engine.py` contains `async def dispose_engine(`
- `backend/src/db/engine.py` contains `def get_session_factory(`
