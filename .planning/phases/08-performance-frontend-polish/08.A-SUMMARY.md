## Key Changes

### Sampled Wall Thickness (context.py)
- `_raycast_sample_threshold()` reads `RAYCAST_SAMPLE_THRESHOLD` env var (default 50000)
- `_compute_wall_thickness_sampled()` samples ~5000 faces via stride, ray-casts them, then propagates to unsampled faces via `scipy.spatial.KDTree` nearest-neighbor lookup
- `_compute_wall_thickness()` delegates to sampled path when face count exceeds threshold

### Mesh Cleanup (analysis_service.py)
- `_force_gc_after_analysis()` reads `FORCE_GC_AFTER_ANALYSIS` env var (default false)
- After `result_dict` is built: `mesh._cache.clear()`, explicit `del ctx, features, geometry`, conditional `gc.collect()`
- Captured `_face_count` and `_verdict` before `del` to avoid NameError in persist block

### Tests
- `test_geometry_context_batching.py`: BuildCounter monkeypatch verifies single `GeometryContext.build` call
- `test_wall_thickness_sampling.py`: env var control, full-vs-sampled correctness (<10% deviation at p95), below-threshold full ray-cast, 200k-face perf benchmark (marked slow)
- `test_mesh_cleanup.py`: cache clear empties dict, del+gc.collect runs without error
