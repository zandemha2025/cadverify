## Implementation Details

### apiClient wrapper (api.ts)
- `apiClient.fetch()` attaches auth headers, extracts rate limits from every response
- `apiClient.fetchJson<T>()` adds 2x retry with 1s/2s backoff for 5xx errors
- 429: toast with Retry-After countdown, no retry
- 5xx: retry then toast + Sentry.captureException
- Network error: "Connection timed out" toast
- Malformed JSON: "Unexpected server response" toast + Sentry.captureMessage
- `getLatestRateLimits()` export for QuotaDisplay consumption
- All 12 exported API functions refactored to use apiClient (except fetchSharedAnalysis, public endpoint)

### Error Boundaries
- Root `error.tsx`: generic "Something went wrong" with Sentry capture, digest display, unstable_retry
- Dashboard `(dashboard)/error.tsx`: contextual "Dashboard error" with area tagging in Sentry extra

### Toast Setup
- sonner Toaster in root layout, position top-right, richColors, closeButton
