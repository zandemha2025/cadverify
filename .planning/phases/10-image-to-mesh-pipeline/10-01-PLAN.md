<action>
Create `backend/src/reconstruction/preprocessing.py` with:

1. `ACCEPTED_IMAGE_TYPES = {"image/jpeg", "image/png", "image/webp"}`
2. `MAX_IMAGE_BYTES = 20 * 1024 * 1024` (20MB)
3. `TARGET_SIZE = 512` (TripoSR native resolution)
4. `BLUR_THRESHOLD = 100.0` (Laplacian variance threshold)

5. `validate_image(file_bytes: bytes, content_type: str) -> None`:
   - Check content_type in ACCEPTED_IMAGE_TYPES, raise ValueError if not
   - Check len(file_bytes) <= MAX_IMAGE_BYTES, raise ValueError if exceeds

6. `detect_blur(image: PIL.Image.Image) -> float`:
   - Convert to grayscale numpy array
   - Compute Laplacian (scipy.ndimage.laplace or cv2.Laplacian approximation via numpy convolution)
   - Return variance of Laplacian. Higher = sharper.

7. `remove_background(image: PIL.Image.Image) -> PIL.Image.Image`:
   - `from rembg import remove`
   - `result = remove(image, model_name="isnet-general-use")`
   - Convert RGBA result to RGB with white background
   - Return RGB image

8. `resize_and_center(image: PIL.Image.Image, target_size: int = TARGET_SIZE) -> PIL.Image.Image`:
   - Calculate scale to fit within target_size maintaining aspect ratio
   - Resize with PIL.Image.LANCZOS
   - Create white target_size x target_size canvas
   - Paste resized image centered
   - Return canvas

9. `preprocess_image(file_bytes: bytes, content_type: str = "image/jpeg") -> tuple[PIL.Image.Image, dict]`:
   - Call validate_image()
   - Open with PIL.Image.open(io.BytesIO(file_bytes)).convert("RGB")
   - Compute blur_score = detect_blur(image)
   - Call remove_background()
   - Call resize_and_center()
   - Return (processed_image, {"blur_score": blur_score, "original_size": (w, h), "warning": "Image may be too blurry" if blur_score < BLUR_THRESHOLD else None})

10. `select_best_image(images: list[tuple[bytes, str]]) -> int`:
    - For each image: compute resolution (w*h) and blur_score
    - Composite score = 0.6 * normalized_resolution + 0.4 * normalized_blur_score
    - Return index of highest scoring image
</action>

<files>backend/src/reconstruction/scoring.py</files>

- backend/src/reconstruction/scoring.py (file being created)

<action>
Create `backend/src/reconstruction/scoring.py` with:

1. Env-var configurable thresholds:
   ```python
   RECON_CONFIDENCE_HIGH = float(os.getenv("RECON_CONFIDENCE_HIGH", "0.7"))
   RECON_CONFIDENCE_LOW = float(os.getenv("RECON_CONFIDENCE_LOW", "0.4"))
   ```

2. `compute_reconstruction_confidence(mesh: trimesh.Trimesh) -> float`:
   - Return 0.0 if mesh has 0 faces
   - Compute 5 weighted metrics (from CONTEXT.md D-09):
     a. Watertight (weight 0.30): 1.0 if `mesh.is_watertight` else 0.0
     b. Degenerate faces (weight 0.20): `1.0 - (count(area_faces < 1e-10) / total_faces)`
     c. Self-intersections (weight 0.20): sample-based check, `1.0 - min(1.0, intersect_ratio)`; wrap in try/except returning 0.5 on failure
     d. Face count adequacy (weight 0.15): sigmoid around range (5000, 200000); 1.0 if in range, linearly decrease outside
     e. Surface smoothness (weight 0.15): mean dot product of adjacent face normals via `mesh.face_adjacency`; higher = smoother
   - Return weighted sum clamped to [0.0, 1.0], rounded to 3 decimal places

3. `confidence_level(score: float) -> str`:
   - Return "high" if score >= RECON_CONFIDENCE_HIGH
   - Return "medium" if score >= RECON_CONFIDENCE_LOW
   - Return "low" otherwise

4. `confidence_message(level: str) -> str | None`:
   - "high": return None (no warning)
   - "medium": return "Reconstruction quality is moderate; results may be less reliable."
   - "low": return "Reconstruction quality is low; consider uploading additional images or a CAD file."
</action>

Create `backend/src/reconstruction/remote_triposr.py` with:

1. `RemoteTripoSR(ReconstructionEngine)` class:
   - `__init__(self)`:
     - `self._api_token = os.getenv("REPLICATE_API_TOKEN")`
     - `self._model_version = os.getenv("TRIPOSR_REPLICATE_MODEL", "stability-ai/triposr")`
     - `self._timeout = int(os.getenv("RECONSTRUCTION_TIMEOUT_SEC", "120"))`
     - If `self._api_token` is None, raise `RuntimeError("REPLICATE_API_TOKEN env var required for remote reconstruction")`

2. `async def reconstruct(self, image_bytes: bytes, params: ReconstructParams) -> ReconstructResult`:
   - Base64 encode image_bytes
   - Use `httpx.AsyncClient` to POST to Replicate API:
     ```
     POST https://api.replicate.com/v1/predictions
     Authorization: Bearer {self._api_token}
     Body: {"version": "...", "input": {"image": "data:image/png;base64,{b64}"}}
     ```
   - Poll prediction status until "succeeded" or "failed" (max self._timeout seconds)
   - On "succeeded": download output mesh URL, read bytes
   - Convert OBJ/GLB output to STL via trimesh if needed
   - Return `ReconstructResult(mesh_bytes=stl_bytes, face_count=..., duration_ms=..., method="triposr_remote")`

3. `async def health_check(self) -> bool`:
   - Check REPLICATE_API_TOKEN is set
   - Optionally ping Replicate API health endpoint
   - Return True/False

4. Error handling:
   - `httpx.TimeoutException` → raise with "Reconstruction timed out after {self._timeout}s"
   - `httpx.HTTPStatusError` (4xx/5xx) → raise with API error detail
   - Replicate "failed" status → raise with failure reason from API
</action>
