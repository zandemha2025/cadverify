2. `RECONSTRUCTION_BACKEND = os.getenv("RECONSTRUCTION_BACKEND", "remote")`

3. `get_reconstruction_engine() -> ReconstructionEngine`:
   - If RECONSTRUCTION_BACKEND == "local": return `LocalTripoSR.load()`

   - Else: raise ValueError(f"Unknown backend: {RECONSTRUCTION_BACKEND}")

   - Create dir `{RECON_BLOB_DIR}/{job_ulid}/input/`

   - Return input directory path

   - Create dir `{RECON_BLOB_DIR}/{job_ulid}/output/`
   - Save mesh_bytes as `mesh.stl`
   - Return full path to mesh.stl

6. `async def create_reconstruction_job(session: AsyncSession, user: AuthedUser, images: list[tuple[bytes, str]], process_types: str | None, rule_pack: str | None) -> Job`:
   - Validate 1-4 images, validate each via `preprocessing.validate_image()`
   - Create Job row: `type='reconstruction'`, `status='queued'`, `params_json={"image_count": len(images), "process_types": process_types, "rule_pack": rule_pack}`
   - Save images to blob storage via `save_reconstruction_images(job.ulid, images)`
   - Enqueue arq task `run_reconstruction_job` with `job.ulid` as argument
   - Return Job

7. `async def get_reconstruction_mesh_path(session: AsyncSession, job_ulid: str, user_id: int) -> str | None`:
   - Load Job by ulid, verify user_id matches
   - Check job.status in ("done", "partial")
   - Return `{RECON_BLOB_DIR}/{job_ulid}/output/mesh.stl` if exists, else None

<files>backend/src/jobs/reconstruction_tasks.py</files>

<action>
Create `backend/src/jobs/reconstruction_tasks.py` with:

`async def run_reconstruction_job(ctx: dict, job_ulid: str) -> dict`:

Following the exact pattern from `run_sam3d_job` in `tasks.py`:

1. `session_factory = get_session_factory()`
2. `async with session_factory() as session:`
3. Load Job by ulid. If not found, log error, return `{"error": "job_not_found"}`
4. Set `job.status = "running"`, `job.started_at = now()`, commit
5. Read images from blob storage: `{RECON_BLOB_DIR}/{job_ulid}/input/`
6. Select best image: `preprocessing.select_best_image(images_with_types)`
7. Preprocess best image: `preprocessing.preprocess_image(image_bytes, content_type)`
8. Get engine: `reconstruction_service.get_reconstruction_engine()`
9. Run inference: `result = await engine.reconstruct(preprocessed_image_bytes, ReconstructParams())`
10. Load reconstructed mesh: `mesh = trimesh.load(io.BytesIO(result.mesh_bytes), file_type="stl")`
11. Compute confidence: `score = scoring.compute_reconstruction_confidence(mesh)`
12. Save mesh to blob: `reconstruction_service.save_reconstruction_mesh(job_ulid, result.mesh_bytes)`
13. **Auto-feed to analysis pipeline:**
    - Extract process_types and rule_pack from `job.params_json`
    - Create a mock `AuthedUser` from job.user_id
    - Call `analysis_service.run_analysis(session, user, mesh_bytes=result.mesh_bytes, filename=f"reconstructed_{job_ulid}.stl", process_types=process_types, rule_pack=rule_pack)`
    - Store returned `analysis.ulid` in result
14. Build result dict:
    ```python
    result_json = {
        "reconstruction": {
            "confidence_score": score,
            "confidence_level": confidence_level(score),
            "confidence_message": confidence_message(confidence_level(score)),
            "face_count": result.face_count,
            "mesh_url": f"/api/v1/reconstructions/{job_ulid}/mesh.stl",
            "duration_ms": result.duration_ms,
            "method": result.method,
        },
        "analysis_id": analysis.ulid if analysis else None,
        "analysis_url": f"/api/v1/analyses/{analysis.ulid}" if analysis else None,
    }
    ```
15. Set `job.status = "done"`, `job.result_json = result_json`, `job.completed_at = now()`, commit
16. On any exception: set `job.status = "failed"`, `job.result_json = {"error": str(e)}`, commit

Error handling: wrap steps 5-14 in try/except. Log full traceback. Never leave job in "running" state.
</action>

<threat_model>
| Threat | Severity | Mitigation |
|--------|----------|------------|
| Unauthenticated access to reconstruction endpoint | high | `Depends(require_api_key)` on all endpoints |
| IDOR: user downloads another user's reconstructed mesh | high | Verify `job.user_id == user.user_id` before serving mesh |
| Path traversal in job_ulid for blob access | medium | ULID format validation (alphanumeric only, fixed length) before constructing file paths |
| Zip bomb via large image upload | medium | MAX_IMAGE_BYTES = 20MB enforced in validate_image() before any processing |
| Rate limit bypass on reconstruction | medium | Existing per-API-key rate limiting (Phase 2) applies to reconstruct endpoint |
</threat_model>
