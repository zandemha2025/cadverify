<objective>
Build the STEP AP242 parser module using OCP's XDE framework (STEPCAFControl_Reader) to extract B-rep geometry and detect PMI presence. This module coexists with the existing step_parser.py (which handles mesh tessellation) and provides the foundation for GD&T extraction in Plan 11-02.
</objective>

<must_haves>
- STEP AP242 file parsed via OCP STEPCAFControl_Reader into XDE document
- B-rep shape hierarchy extracted and traversable
- PMI presence detection (has_pmi flag based on DimTolTool label count)
- AP214 files read successfully for geometry-only (no PMI, no error)
- Graceful fallback when OCP XDE modules are not available
</must_haves>

<action>
Create `backend/src/parsers/step_ap242_parser.py` with:

1. Feature-gating at module level (same pattern as step_parser.py):
   ```python
   _HAS_XDE = False
   try:
       from OCP.STEPCAFControl import STEPCAFControl_Reader
       from OCP.TDocStd import TDocStd_Document
       from OCP.XCAFApp import XCAFApp_Application
       from OCP.XCAFDoc import XCAFDoc_DimTolTool, XCAFDoc_ShapeTool
       from OCP.TCollection import TCollection_ExtendedString
       from OCP.IFSelect import IFSelect_RetDone
       _HAS_XDE = True
   except ImportError:
       pass
   ```

2. `is_ap242_supported() -> bool` function returning `_HAS_XDE`.

3. `AP242Document` dataclass:
   - `doc: Any` (TDocStd_Document reference)
   - `shape_tool: Any` (XCAFDoc_ShapeTool)
   - `dim_tol_tool: Any` (XCAFDoc_DimTolTool)
   - `has_pmi: bool` (True if dim_tol_tool has GDT labels)
   - `shape_labels: list` (top-level shape labels)

4. `parse_ap242(file_path: str | Path) -> AP242Document` function:
   - Validate file exists and has .step/.stp extension.
   - Create `XCAFApp_Application.GetApplication()`.
   - Create `TDocStd_Document(TCollection_ExtendedString("XDE"))`.
   - Create `STEPCAFControl_Reader()`, call `reader.ReadFile(str(path))`.
   - Check return value equals `IFSelect_RetDone`, raise `ValueError` if not.
   - Call `reader.Transfer(doc)`.
   - Get `ShapeTool` and `DimTolTool` from document main label.
   - Get GDT labels via `dim_tol_tool.GetGDTLabels()` — set `has_pmi = len(labels) > 0`.
   - Get top-level shape labels via `shape_tool.GetFreeShapes()`.
   - Return `AP242Document`.

5. `parse_ap242_from_bytes(data: bytes, filename: str) -> AP242Document` function:
   - Same temp-file pattern as `step_parser.py:parse_step_from_bytes`.
   - Write bytes to temp file with `mode=0o600`, parse, clean up in finally block.

All imports guarded by `_HAS_XDE`. If not available, functions raise `RuntimeError("AP242 parsing requires OCP XDE modules")`.
